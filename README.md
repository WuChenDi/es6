# gulp es6-caipiao

- Study es6

## 项目运行（3000）
通过npm安装本地服务第三方依赖模块(需要已安装[Node.js](https://nodejs.org/en/))

```
cd es6

npm install 或 cnpm install(个人比较喜欢使用后者，下载依赖模块速度较快)

gulp --watch

```
